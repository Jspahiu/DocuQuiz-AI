# DocuQuiz-AI
## Work in progress, it is in very early stages!
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.12.7-brightgreen?)
![Hackatime](https://hackatime-badge.hackclub.com/U0966SKR40M/DocuQuiz%20AI) 


DocuQuiz AI is a program powered by AI that can make a quiz based on anything you have! All you need is one picture!

It can:
- Train ChatGPT with your own PDF or pictures
- Create a quiz in PDF or CSV File
- Can use CSV file for Blooket!
